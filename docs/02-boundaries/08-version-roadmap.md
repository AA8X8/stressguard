План развития StressGuard построен с учётом B2B2C-модели, ключевых технологических особенностей (интеграция с носимыми устройствами, ML-модель, геймификация) и обратной связи от пилотных клиентов. Разработка ведётся итеративно, с регулярными релизами новых функций.

## Версия 1.0 (MVP) – 6 месяцев

**Цель:** Запустить минимально жизнеспособный продукт для проверки гипотез и привлечения первых пилотных организаций/вузов.

**Основные функции:**
- Регистрация через email/социальные сети.
- Интеграция с календарями (Google Calendar, Apple Calendar) для сбора событий.
- Интеграция с носимыми устройствами через Google Fit (Android) и Apple Health (iOS) для получения пульса, HRV, сна, активности.
- Ручной ввод настроения и усталости с начислением баллов (плюшки).
- ML-модель для расчёта стресс-индекса (низкий/средний/высокий) на основе комбинации объективных и субъективных данных.
- Отображение текущего стресс-индекса и истории за 7 дней.
- База из 20–30 универсальных рекомендаций, доставляемых по правилам (высокий стресс, плохой сон).
- Простая система мотивации: баллы, уровни, виртуальные награды.
- Базовый веб-дашборд для HR/деканата: средний стресс по организации, количество активных пользователей, динамика.
- Админ-панель для управления организациями и контентом рекомендаций.

**Результат:** Проведение пилотного проекта в 2–3 организациях/вузах, сбор обратной связи, валидация ключевых метрик (удержание, активность, удовлетворённость).

## Версия 1.1 – через 3 месяца после MVP

**Цель:** Улучшить продукт на основе обратной связи пилотов, расширить интеграции и повысить вовлечённость.

**Новые функции и улучшения:**
- Поддержка календаря Microsoft Outlook (интеграция с Microsoft Graph).
- Добавление интеграции с Garmin и Mi Band (через открытые API) для расширения аудитории.
- Улучшение ML-модели: учёт большего количества признаков (например, вариабельность пульса в течение дня), дообучение на данных пилотов.
- Расширение базы рекомендаций до 50+ советов, возможность добавлять корпоративные рекомендации (через HR-дашборд).
- Улучшение системы плюшек: появление еженедельных челленджей (например, «7 дней без пропусков ввода настроения»).
- Добавление экспорта отчётов для HR в формате PDF (базовая версия).
- Настройка уведомлений: возможность выбора времени «не беспокоить».

**Результат:** Повышение активности пользователей, удовлетворённости HR-клиентов, подготовка к масштабированию.

## Версия 2.0 – через 6–8 месяцев после MVP

**Цель:** Масштабирование продукта, переход к полноценной коммерческой модели, запуск партнёрской сети реальных бонусов.

**Новые функции и улучшения:**
- **Партнёрская программа:** интеграция с реальными компаниями (кофейни, фитнес-клубы, книжные сервисы) для обмена баллов на скидки и промокоды. Автоматическая выдача бонусов через API партнёров.
- **Расширенная аналитика для HR:**
  - Тепловые карты стресса по дням недели и времени суток.
  - Сравнение отделов/курсов между собой.
  - Прогнозирование риска выгорания на уровне группы (анонимно).
  - Гибкие отчёты с настраиваемыми периодами.
- **Улучшенная ML-модель:**
  - Персонализация под конкретного пользователя (fine-tuning на его данных с сохранением приватности).
  - Прогноз стресса на следующий день на основе календаря и истории.
- **Геймификация 2.0:**
  - Командные челленджи внутри организации (например, «какой отдел снизит стресс больше всех»).
  - Рейтинги (анонимные) и достижения с реальными призами.
- Мобильное приложение для watchOS / Wear OS (базовый функционал: отображение стресс-индекса, быстрый ввод настроения).
- SSO-интеграция для корпоративных клиентов (вход через корпоративный Google/Microsoft).

**Результат:** Выход на самоокупаемость, рост числа платных клиентов, укрепление позиций на рынке B2B wellness-решений.

## Версия 3.0 – через 12–18 месяцев после MVP

**Цель:** Стать лидером в сегменте корпоративных и образовательных решений для ментального здоровья, внедрение инновационных
технологий.

**Новые функции и улучшения:**
- **Интеграция с дополнительными источниками данных:**
  - Анализ текстовых заметок с помощью NLP (выявление эмоционального окраса, ключевых стрессоров).
  - Опциональное подключение данных об окружающей среде (погода, уровень шума, геолокация) для более точного контекста.
- **Предиктивная аналитика для HR:**
  - Модель, предсказывающая вероятность выгорания конкретного сотрудника (с его согласия) для проактивного вмешательства психологов.
- **Расширение экосистемы:**
  - API для сторонних разработчиков (возможность интеграции StressGuard в корпоративные порталы и приложения).
  - Партнёрство со страховыми компаниями (скидки на ДМС при регулярном использовании).
- **Медицинская опция (опционально):**
  - Получение лицензии на медицинское изделие (для работы с клиниками и врачами), строгое соответствие регуляторным требованиям.
- **Социальные функции:**
  - Группы поддержки, анонимные обсуждения, совместные медитации (в рамках корпоративной этики).
- **Интернационализация:**
  - Поддержка основных европейских и азиатских языков, адаптация под локальные рынки.

**Результат:** Устойчивый бизнес с диверсифицированными источниками дохода (B2B-лицензии, партнёрские программы, API-доступ), высокая лояльность пользователей.

## Управление изменениями

- Бэклог продукта ведётся в системе (Jira/Trello), приоритеты пересматриваются ежемесячно с учётом обратной связи клиентов и бизнес-целей.
- Каждая версия включает не только новые функции, но и техническое совершенствование (оптимизация производительности, безопасности).
- Внедрение критически важных улучшений (например, безопасность, масштабирование) может выполняться вне очереди.

> План развития является гибким и может корректироваться по результатам пилотных проектов и изменений рыночной ситуации.
