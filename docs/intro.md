**StressGuard** — B2B2C-платформа для проактивного мониторинга и управления стрессом.  
Продукт предназначен для сотрудников компаний и студентов вузов. В автоматическом режиме агрегирует объективные данные (пульс, активность и события из календаря) и субъективные отметки пользователя (настроение, усталость, сон), анализирует их с помощью ML‑модели и предоставляет персонализированные рекомендации по снижению стресса.

## Основные функции

- **Регистрация и интеграция** – вход через email/соцсети, подключение календарей (Google, Apple, Outlook) и носимых устройств (Google Fit / Apple Health)
- **Автоматический сбор данных** – пульс, шаги, плотность встреч из календаря
- **Ручной ввод самочувствия** – настроение, усталость, параметры сна
- **Расчёт стресс-индекса (ML)** – ежедневный показатель (0–100) на основе комбинации объективных и субъективных признаков
- **Персонализированные рекомендации** – база советов по пяти категориям (движение, питание, отдых, планирование, общение)
- **DMN (Decision Model and Notation)** – расчёт начисления баллов за полный цикл (ввод настроения, усталости, сна + оценка рекомендаций), а также формирование событий для HR
- **HR‑дашборд** – агрегированная анонимная статистика по организации/вузу: средний стресс, активность пользователей, список достигших цели для выдачи бонусов

## Надёжность и гарантии

- **Расчёт стресс-индекса** – выполняется в фоновом режиме, время расчёта не более 5 минут на пользователя за день.
- **Доставка рекомендаций** – после расчёта стресс-индекса пользователь получает персонализированные советы в мобильном приложении.
- **Отказоустойчивость** – при недоступности внешних API (Google Fit, календарь) система продолжает работу, используя доступные данные; точность расчёта снижается, но пользовательский процесс не блокируется.
- **Асинхронные уведомления** – уведомления HR‑специалисту о достижении пользователем цели доставляются через RabbitMQ; сбои в отправке не влияют на основной процесс.
